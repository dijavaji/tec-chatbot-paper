# Documentación de Historias de Usuario - Sistema Smart de Chatbot

Esta sección detalla las historias de usuario del Sistema Smart de Chatbot, extraídas de la propuesta técnica. Cada historia de usuario describe una funcionalidad desde la perspectiva del usuario final, incluyendo su identificador, descripción, prioridad y estimación. Estos elementos son fundamentales para la planificación y el desarrollo ágil del proyecto.
